--ROS install manual-- (ver 1.0)

1. You must compile with the following command:

chmod 700 ROS_Install.sh
./ROS_Install.sh

2. This script is only available for Ubuntu version 16.04.

3. The ROS version that you download to the script is "Kenetic".
